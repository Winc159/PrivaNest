# 统一媒体文件布局优化

## 🎯 优化目标

实现**图片、视频、文件夹**三类文件的视觉统一和面积协调，打造类似 macOS 相册的专业体验。

---

## ✨ 核心改进

### 1. **三类文件的统一处理**

| 文件类型 | 方向检测 | 容器比例 | 显示策略 | 视觉效果 |
|---------|---------|---------|---------|---------|
| **横拍图** | ✅ 自动检测 | 1.6 : 1 | contain | 宽扁型，上下留白 |
| **竖拍图** | ✅ 自动检测 | 0.75 : 1 | contain | 高挑型，左右留白 |
| **正方形图** | ✅ 自动检测 | 1 : 1 | cover | 标准方块 |
| **视频文件** | ⚠️ 默认横拍 | 1.6 : 1 | 渐变背景 + 图标 | 宽扁型，绿色渐变 |
| **文件夹** | ❌ 固定正方 | 1 : 1 | 渐变背景 + 图标 | 标准方块，紫色渐变 |

---

## 🔧 技术实现

### 1. **图片方向智能检测**

```typescript
const updateImageOrientation = (width: number, height: number) => {
  const aspectRatio = width / height
  
  if (aspectRatio > 1.2) {
    imageOrientation.value = 'landscape'  // 横拍
  } else if (aspectRatio < 0.8) {
    imageOrientation.value = 'portrait'   // 竖拍
  } else {
    imageOrientation.value = 'square'     // 正方
  }
}
```

**触发时机：**
- NImage 组件：`@load` 事件获取 `naturalWidth/naturalHeight`
- Canvas 缩略图：生成时计算并存储到 `dataset.orientation`

---

### 2. **视频文件特殊处理**

#### 问题
视频文件没有实际加载封面图（只显示图标），无法自动检测方向。

#### 解决方案
```typescript
// 判断是否为视频文件
const isVideoFile = computed(() => {
  const ext = props.file.ext?.toLowerCase()
  return ['mp4', 'avi', 'mov', 'mkv', 'wmv', 'flv', 'webm'].includes(ext)
})

// 视频默认使用横拍比例（大多数视频是横屏）
if (isVideoFile.value) {
  containerAspectRatio.value = 1.6
}
```

**CSS 样式：**
```scss
&.video-wrapper {
  display: flex;
  align-items: center;
  justify-content: center;
  background: linear-gradient(135deg, #18a058 0%, #0f9d58 100%);
  
  .video-icon {
    filter: drop-shadow(0 4px 8px rgba(24, 160, 88, 0.4));
  }
}
```

**效果：**
- ✅ 默认横拍比例 (1.6:1)
- ✅ 绿色渐变背景
- ✅ 居中显示播放图标
- ✅ 悬停时图标放大

## 💡 设计理念

### 1. **尊重内容形态**
- 横图横着放，竖图竖着放
- 视频默认横拍（符合大多数场景）
- 文件夹保持正方（视觉稳定）

### 2. **面积统一原则**
- 移除文件夹的 2×2 特殊待遇
- 所有项目占用相近面积
- 网格排列更紧凑

### 3. **渐进增强**
- 有封面的显示封面（图片）
- 无封面的显示图标 + 渐变背景（视频、文件夹）
- 悬停时有微交互反馈

## 🔍 特殊情况处理

### 1. **竖屏视频**
虽然少见，但如果存在竖屏视频（如抖音格式）：
- 当前方案：按横屏处理 (1.6:1)
- 未来优化：读取视频元数据，自动检测方向

### 2. **未知格式文件**
既不是图片也不是视频：
- 默认显示灰色图标
- 使用正方形容器 (1:1)

### 3. **缩略图加载失败**
```typescript
// 错误处理：如果是视频，设置默认的 landscape 标识
if (src.includes('/api/media/file')) {
  const ext = extractExtension(path)
  if (isVideoExt(ext)) {
    canvas.dataset.orientation = 'landscape'
  }
}
```

## 📈 性能优化

### 1. **懒加载策略**
- 只加载可视区域内的缩略图
- IntersectionObserver 监听滚动

### 2. **智能压缩**
- < 500KB：直接加载原图
- ≥ 500KB：Canvas 压缩到 300×300

### 3. **透明背景**
```scss
background: transparent;
```
减少绘制开销，更清爽。
