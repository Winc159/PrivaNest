# Library 页面 UI 优化说明

## 🎨 优化概览

本次重构重点优化了媒体库页面的视觉体验和布局，特别针对**横拍和竖拍照片**的混合展示场景进行了专门设计。

---

## ✨ 主要改进

### 1. **横竖屏照片自适应布局**

#### 问题
- 传统固定高度的缩略图会裁剪竖拍照片的顶部和底部
- 横拍照片在正方形容器中浪费大量空间

#### 解决方案
```scss
.thumbnail-wrapper {
  height: 200px; // 统一容器高度
  
  .thumbnail-canvas,
  .thumbnail {
    width: 100%;
    height: 100%;
    object-fit: contain; // ✅ 关键：保持宽高比，完整显示
  }
}
```

**效果对比：**

| 方案 | 横拍照片 (16:9) | 竖拍照片 (9:16) | 空间利用率 |
|------|----------------|----------------|-----------|
| **旧方案** (object-fit: cover) | ❌ 裁剪上下边缘 | ❌ 裁剪左右两侧 | 低 |
| **新方案** (object-fit: contain) | ✅ 完整显示，两侧留白 | ✅ 完整显示，上下留白 | 高 |

### 2. **响应式网格系统**

使用 CSS Grid 实现真正的响应式布局：

```scss
.grid-container {
  display: grid;
  gap: 20px;
  
  // 移动端：2 列
  grid-template-columns: repeat(2, 1fr);
  
  // 平板：3-5 列
  @media (min-width: 600px) { grid-template-columns: repeat(3, 1fr); }
  @media (min-width: 768px) { grid-template-columns: repeat(4, 1fr); }
  @media (min-width: 900px) { grid-template-columns: repeat(5, 1fr); }
  @media (min-width: 1200px) { grid-template-columns: repeat(6, 1fr); }
  @media (min-width: 1600px) { grid-template-columns: repeat(7, 1fr); }
}
```

**断点说明：**
- 📱 < 600px: 2 列（手机）
- 📱 600-768px: 3 列（大手机/小平板）
- 📱 768-900px: 4 列（平板）
- 💻 900-1200px: 5 列（小屏笔记本）
- 💻 1200-1600px: 6 列（标准桌面）
- 🖥️ > 1600px: 7 列（大屏显示器）

### 3. **卡片视觉升级**

#### 3.1 整体设计
- **圆角**: 从 `4px` → `12px`（更现代）
- **阴影**: 增加多层阴影，悬停时更明显
- **动画**: 使用 `cubic-bezier(0.4, 0, 0.2, 1)` 缓动（更流畅）

#### 3.2 悬停效果
```scss
&:hover {
  transform: translateY(-6px) scale(1.02); // 上浮 + 微放大
  box-shadow: 0 12px 24px rgba(0, 0, 0, 0.15); // 加深阴影
  
  .thumbnail-wrapper {
    .thumbnail-canvas,
    .thumbnail {
      transform: scale(1.05); // 图片微放大
    }
  }
}
```

#### 3.3 文件夹渐变
```scss
background: linear-gradient(135deg, #667eea 0%, #764ba2 100%);
```
- 从紫色到深紫的渐变
- 半透明白色遮罩层
- 图标添加投影效果

#### 3.4 文件卡片
- **白色背景** + 轻微阴影
- **独立信息区域**（与缩略图分离）
- **文件名**: 最多显示 2 行，超出省略
- **文件大小**: 灰色小字显示

### 4. **列表模式优化**

#### 网格模式 vs 列表模式

| 属性 | 网格模式 | 列表模式 |
|------|---------|---------|
| 缩略图尺寸 | 200px 高 | 80px 高 |
| 布局方向 | 垂直 | 水平 |
| 信息位置 | 图片下方 | 图片右侧 |
| 适用场景 | 浏览封面墙 | 快速查找文件 |

**列表模式样式：**
```scss
&.list {
  .card-content {
    flex-direction: row; // 横向布局
    
    .thumbnail-wrapper {
      width: 120px;
      height: 80px;
      border-radius: 8px 0 0 8px;
    }
    
    .file-info {
      flex-direction: row; // 信息横向排列
      gap: 16px;
    }
  }
}
```

## 📊 性能优化

### 图片加载策略
1. **懒加载**: 使用 IntersectionObserver
2. **缩略图生成**: 仅对 >500KB 的大图处理
3. **压缩质量**: 80% JPEG（平衡清晰度与体积）

### CSS 性能
- 使用 `transform` 而非 `top/left`（GPU 加速）
- 避免过度使用 `box-shadow`
- 合理使用 `will-change`
